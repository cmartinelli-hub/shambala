Instruções de Configuração

1. Plugue o Pendrive

Conecte o pendrive ou HD externo ao servidor.

2. Identifique o Dispositivo

Abra um terminal e execute:

lsblk

ou

dmesg | tail -20

Procure por algo como /dev/sdb1

3. Permissões Sudo (uma vez)

Para que o sistema possa montar/desmontar, execute como root:

echo "shambala ALL=(ALL) NOPASSWD: /usr/bin/mount, /usr/bin/umount" | sudo tee -a /etc/sudoers.d/shambala

4. Configure no Formulário Acima

  • Dispositivo: /dev/sdb1 (ajuste conforme seu hardware)
  • Ponto de Montagem: /mnt/pendrive (ou outro local)
  • Clique em "Testar Conexão" para validar

5. Backup Automático (Opcional)

Ative a opção "Ativar backup automático" e escolha um horário. O sistema criará um cron job para executar o backup automaticamente.
