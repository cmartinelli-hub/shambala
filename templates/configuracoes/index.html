Backup Remoto

Envie cópias automáticas do backup para um servidor remoto via SSH/SCP ao encerrar o dia.

🔑 Configurando SSH por chave

  1. Gere uma chave SSH (se não existir): 
    ssh-keygen -t ed25519 -C "shambala-backup"
  2. Copie a chave pública: 
    ssh-copy-id -i ~/.ssh/id_ed25519.pub usuario@servidor
    Substitua usuario e servidor pelos dados do servidor remoto.
  3. Teste: 
    ssh usuario@servidor "mkdir -p ~/backups/shambala"
  4. Configure abaixo os dados de acesso.
